PROJECTS
Books Scraper
- scrapes book prices from Books to Scrape website
- saves the extracted prices with timestamps to a CSV file

Country Scraper
- scrapes country names, capitals, and populations
- saves the extracted data to a CSV file

Quotes Scraper 
- scrapes quotes and their authors from Quotes to Scrape website
- saves the extracted data to a CSV file

Weather API
- retrieves current weather data from the open-mateo API
- extracts the temperature, humidity, and wind speed
- saves the weather data to a CSV file
